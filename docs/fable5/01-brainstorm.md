# Step 01 — Brainstorm

| | |
|---|---|
| Tool | Sonnet 5 |
| Role | Brainstorm Buddy |
| Cost tier | Low |

## What to do

Brain-dump the entire idea into a chat session. Ask questions, conduct research,
explore edge cases, and iterate until there's a deep, comprehensive understanding of
the project scope. This is the ideation phase — no coding, just exploration.

## Key requirement

Make sure the Linear MCP connector is connected so Sonnet 5 can interact with the
project management system directly. This works from a normal chat, mobile included.

## Prompt template

```
I want to build [PROJECT DESCRIPTION]. Let's do a deep brainstorm session.
Ask me clarifying questions, challenge my assumptions, suggest features I
might have missed, and help me fully scope this project. I have Linear MCP
connected — keep that in mind for how we'll structure this later.
```

## Output

A clear, agreed-upon scope. Nothing gets written to Linear yet — that's Step 02.
